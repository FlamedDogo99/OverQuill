VerQuill

Interactive LaTeX editing in Overleaf, using a custom build of MathQuill

Keyboard Shortcuts

Open Editor: Alt + t - Open the in-editor MathQuill editor

  • Must be editing inside the Overleaf code editor
  • Shortcut can be configured in the extension's options menu

LaTeX Command: \ - Invoke one of MathQuill's LaTeX commands

  • Must be editing inside the MathQuill editor
  • Suggestions will be displayed below the editor for all registered LaTeX commands and Environment commands

Add Matrix Row: Command + Up Arrow - Add a row to the current Matrix

  • Must be editing inside the MathQuill editor and inside a matrix environment

Add Matrix Column: Command + Right Arrow - Add a column to the current Matrix

  • Must be editing inside the MathQuill editor and inside a matrix environment

Remove Matrix Row: Command + Down Arrow - Remove a row to the current Matrix

  • Must be editing inside the MathQuill editor and inside a matrix environment

Add Matrix Column: Command + Left Arrow - Remove a column to the current Matrix

  • Must be editing inside the MathQuill editor and inside a matrix environment

Use Suggested LaTeX Command: Tab - Call a suggested LaTeX command

  • Must be editing inside the MathQuill editor and writing a command
  • By default, the first suggestion will be used to complete the command
  • Other suggestions can be navigated through with Up Arrow and Down Arrow, or by mouse
  • If no suggestions exist, it will default to MathQuill's default behavior

Use Written LaTeX Command: Space - Call the LaTeX command without suggestions

  • Must be editing inside the MathQuill editor and writing a command
  • Uses MathQuill's default behavior for LaTeX commands

Insert LaTeX Into Overleaf Return - Insert LaTeX into the Overleaf document

  • Must be editing inside the MathQuill editor

Exit Editor Escape - Exit the MathQuill Editor

  • Must be editing inside the MathQuill editor
  • Any LaTeX written inside the editor will be discarded
